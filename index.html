Cara Menjalankan

Aplikasi ini dibangun dengan React Native dan Expo. Untuk menjalankannya:

# Clone repository
git clone https://github.com/username/GameImpostor.git

# Masuk ke folder proyek
cd GameImpostor

# Install dependencies
npm install

# Jalankan aplikasi
npx expo start

# Scan QR code dengan Expo Go di smartphone

Atau build untuk production:

# Build untuk Android
npx expo run:android

# Build untuk iOS (hanya di Mac)
npx expo run:ios
